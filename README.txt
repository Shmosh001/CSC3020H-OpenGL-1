Description: An stl image viewer that allows the user to open an .stl file and view it. The program allows the 3D image to be scaled, rotated or transformed along the x,y,z axis.

Controls: 
	Rotation: Press 'r' to toggle rotation mode. Each time 'r' is pressed the axis for rotation is changed. To rotate the image, use the mouse scroller.
	
	Transformation: Press 't' to toggle transformation mode. Each time 't' is pressed the axis for transformation is changed. To transform the image, use the mouse scroller.

	Scale: Press 's' to toggle sclae mode. To scale the image, use the mouse scroller.
